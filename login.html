사용 전 확인

  • Supabase SQL Editor에서 supabase-study-data.sql 내용을 먼저 실행해야 학습 기록 저장이 됩니다.
  • 이메일 확인 설정이 켜져 있으면 가입 후 메일 인증을 해야 로그인됩니다.
  • 로그인하지 않아도 기존처럼 현재 브라우저에는 임시 기록이 저장됩니다.
