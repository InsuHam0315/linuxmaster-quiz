제2과목: 리눅스 운영 및 관리

명령어, 파일 관리, 디렉터리 구조, vi, 권한, 프로세스 기초가 핵심이며 실제 1차 시험에서 가장 많은 비중을 차지합니다.

A. 기본 명령어

명령어의미자주 쓰는 옵션시험 함정
pwd현재 작업 디렉터리 출력-현재 위치 확인은 cd가 아니라 pwd이다.
cd디렉터리 이동cd -, cd ~인자 없이 실행하면 홈으로 이동한다.
ls파일 목록 출력-l, -a, -h, -R숨김 파일은 -a가 있어야 보인다.
mkdir디렉터리 생성-p상위 디렉터리까지 만들 때 -p를 쓴다.
rmdir빈 디렉터리 삭제-p비어 있지 않으면 삭제하지 못한다.
cp파일 복사-r, -p, -i디렉터리 복사는 -r이 필요하다.
mv이동 또는 이름 변경-i, -f같은 파일시스템에서는 이름 변경처럼 동작할 수 있다.
rm파일 삭제-r, -f, -i디렉터리와 내부 삭제는 rm -r이다.
touch빈 파일 생성 또는 시간 갱신-t내용을 출력하는 명령이 아니다.
cat파일 내용 출력과 연결-n긴 파일 탐색은 more/less가 더 적합하다.
more페이지 단위 출력-앞으로 넘겨 보는 기본 페이저이다.
less향상된 페이지 단위 출력-N앞뒤 이동이 more보다 편하다.
head앞부분 출력-n기본은 앞 10줄이다.
tail뒷부분 출력-n, -f로그 실시간 확인은 tail -f이다.
wc줄, 단어, 바이트 수 계산-l, -w, -cwc -l은 줄 수이다.
sort정렬-r, -n숫자 정렬은 -n이 필요하다.
grep문자열 검색-i, -n, -r, -v파일 이름 검색이 아니라 내용 검색이다.
find파일과 디렉터리 검색-name, -type, -size이름 조건은 find 경로 -name 패턴 형식이다.

B. ls 주요 옵션

  • -l: 자세한 정보 출력
  • -a: 숨김 파일 포함
  • -al: 숨김 파일까지 자세히 출력
  • -h: 용량을 사람이 읽기 쉬운 단위로 표시
  • -R: 하위 디렉터리까지 재귀 출력
  • -d: 디렉터리 자체 정보 출력
  • -i: inode 번호 출력

C. 파일과 디렉터리 관리

절대경로는 `/`에서 시작하는 전체 경로이고, 상대경로는 현재 위치를 기준으로 표현합니다. 현재 디렉터리는 `.`, 상위 디렉터리는 `..`, 홈 디렉터리는 `~`로 표시합니다.

  • 숨김 파일은 이름이 `.`으로 시작합니다.
  • 파일 복사는 `cp`, 이동과 이름 변경은 `mv`, 삭제는 `rm`을 사용합니다.
  • 빈 디렉터리는 `rmdir`로 삭제할 수 있습니다.
  • 내용이 있는 디렉터리는 `rm -r`로 재귀 삭제합니다.

D. 리다이렉션과 파이프 기초

기호의미
>표준출력을 파일로 저장하며 기존 내용을 덮어쓴다.
>>표준출력을 파일 끝에 추가한다.
<파일 내용을 표준입력으로 사용한다.
2>표준에러만 파일로 보낸다.
2>&1표준에러를 표준출력과 같은 대상으로 합친다.
|앞 명령의 표준출력을 뒤 명령의 표준입력으로 전달한다.

표준입력은 키보드나 파일에서 들어오는 입력, 표준출력은 정상 결과, 표준에러는 오류 메시지 흐름입니다.

E. vi 편집기 기초

  • 명령 모드: 이동, 삭제, 복사 같은 기본 조작을 수행합니다.
  • 입력 모드: 실제 문자를 입력합니다. `i`, `a`, `o`로 진입하고 `ESC`로 명령 모드로 돌아갑니다.
  • ex 모드: `:`로 시작하는 저장, 종료, 치환 명령을 실행합니다.
  • `:w` 저장, `:q` 종료, `:wq` 저장 후 종료, `:q!` 저장하지 않고 종료
  • `dd` 현재 줄 삭제, `yy` 현재 줄 복사, `p` 붙여넣기, `u` 실행 취소
  • `/검색어` 아래 방향 검색, `n` 다음 검색, `N` 반대 방향 검색

F. 권한 기초

`ls -l`의 권한 표시는 파일 종류 뒤에 user, group, others 순서로 `rwx`가 반복됩니다. r은 읽기 4, w는 쓰기 2, x는 실행 1입니다.

  • `chmod 755`: rwxr-xr-x
  • `chmod 644`: rw-r--r--
  • `chown`: 소유자 변경, `chgrp`: 그룹 변경
  • `umask`: 새 파일과 디렉터리 생성 시 기본 권한에서 제거할 권한
  • 파일 기본 권한은 666, 디렉터리 기본 권한은 777에서 umask를 뺀다고 기억합니다.

G. 프로세스 기초

  • 프로세스는 실행 중인 프로그램이며 PID로 구분합니다.
  • `ps`는 프로세스 목록, `top`은 실시간 프로세스 상태를 보여줍니다.
  • `kill`은 PID에 시그널을 보내며 종료 요청에 자주 사용됩니다.
  • 명령 뒤에 `&`를 붙이면 백그라운드로 실행합니다.
  • `jobs`는 작업 목록, `fg`는 포어그라운드 전환, `bg`는 백그라운드 재개입니다.
  • `Ctrl+C`는 포어그라운드 프로세스 종료 요청, `Ctrl+Z`는 일시정지입니다.

H. 셸 기초

셸은 사용자 명령을 해석해 커널에 전달하는 프로그램입니다. 리눅스 1차 수준에서는 bash, 환경변수, PATH, alias, history를 중심으로 봅니다.

  • `echo`: 문자열이나 변수 값 출력
  • `export`: 셸 변수를 환경변수로 내보냄
  • `PATH`: 명령어 검색 경로
  • `alias`: 명령어 별칭 설정
  • `history`: 명령 실행 이력 확인
  • `.bash_profile`은 로그인 셸 시작 시, `.bashrc`는 대화형 bash 실행 시 주로 읽습니다.

I. 패키지 관리 기초

계열도구핵심
Red Hat 계열rpm, yum, dnfrpm은 개별 패키지, yum/dnf는 저장소 기반 관리
Debian 계열dpkg, apt, apt-getdpkg는 개별 패키지, apt/apt-get은 저장소 기반 관리

J. 압축 기초

  • `tar`: 여러 파일을 하나의 아카이브로 묶거나 푼다.
  • `gzip`: `.gz`, `bzip2`: `.bz2`, `xz`: `.xz` 압축에 사용한다.
  • `zip`, `unzip`: zip 형식 압축과 해제에 사용한다.
  • `tar -cvf file.tar dir`: tar 아카이브 생성
  • `tar -xvf file.tar`: tar 아카이브 해제
  • `tar -czf file.tar.gz dir`: gzip 압축 tar 생성
  • `tar -xzf file.tar.gz`: gzip 압축 tar 해제

K. 시험 함정 포인트

  • ls -a는 숨김 파일까지 출력한다.
  • ls -l은 자세한 정보를 출력한다.
  • rm -r은 디렉터리와 내부 내용을 재귀적으로 삭제한다.
  • >는 덮어쓰기, >>는 추가이다.
  • 2>&1은 표준에러를 표준출력으로 합친다.
  • vi에서 :q!는 저장하지 않고 종료한다.
  • chmod 755는 rwxr-xr-x이다.
  • 파일 기본 권한은 666에서 umask를 뺀다.
  • Ctrl+C는 포어그라운드 프로세스 종료 요청이다.
  • Ctrl+Z는 일시정지이다.
  • apt update는 패키지 목록 갱신이다.
  • tar에서 c는 생성, x는 해제이다.

L. 예상문제

1. 현재 디렉터리의 절대경로를 출력하는 명령은?

  1. cd
  2. pwd
  3. ls
  4. mkdir
정답: 2. pwd는 현재 작업 디렉터리를 출력한다.

2. 숨김 파일까지 출력하는 ls 옵션은?

  1. -l
  2. -a
  3. -h
  4. -i
정답: 2. 숨김 파일은 이름이 .으로 시작하며 ls -a로 확인한다.

3. 비어 있지 않은 디렉터리를 삭제할 때 주로 쓰는 명령은?

  1. rmdir dir
  2. rm -r dir
  3. touch dir
  4. cat dir
정답: 2. rm -r은 디렉터리 내부까지 재귀적으로 삭제한다.

4. 기존 파일 끝에 출력을 추가하는 기호는?

  1. >
  2. >>
  3. <
  4. |
정답: 2. >>는 append, >는 덮어쓰기이다.

5. 앞 명령의 출력을 뒤 명령의 입력으로 넘기는 기호는?

  1. |
  2. 2>
  3. <
  4. &
정답: 1. 파이프는 표준출력을 다음 명령의 표준입력으로 연결한다.

6. vi에서 저장 후 종료하는 명령은?

  1. :q!
  2. :wq
  3. dd
  4. yy
정답: 2. :wq는 저장하고 종료한다.

7. vi에서 현재 줄을 삭제하는 명령은?

  1. dd
  2. yy
  3. p
  4. u
정답: 1. dd는 현재 줄 삭제이다.

8. chmod 644의 권한 형태는?

  1. rwxr-xr-x
  2. rw-r--r--
  3. rwx------
  4. rw-rw-rw-
정답: 2. 6은 rw-, 4는 r--이므로 rw-r--r--이다.

9. 실행 중인 프로세스를 실시간으로 확인하는 명령은?

  1. top
  2. pwd
  3. grep
  4. chmod
정답: 1. top은 프로세스 상태를 실시간으로 보여준다.

10. 백그라운드 작업 목록을 확인하는 명령은?

  1. jobs
  2. fg
  3. bg
  4. kill
정답: 1. jobs는 현재 셸의 작업 목록을 보여준다.

11. Debian 계열 패키지 관리 도구는?

  1. rpm
  2. yum
  3. apt-get
  4. dnf
정답: 3. apt-get은 Debian/Ubuntu 계열에서 사용한다.

12. gzip 압축 tar 파일을 해제하는 명령으로 알맞은 것은?

  1. tar -cvf file.tar
  2. tar -xzf file.tar.gz
  3. gzip file.tar.gz
  4. zip file.tar.gz
정답: 2. x는 해제, z는 gzip, f는 파일 지정을 의미한다.
