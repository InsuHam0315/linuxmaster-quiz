제1과목: 리눅스 일반

암기 비중이 높고, 리눅스의 개념·역사·특징·배포판·설치 기본을 묻는 영역입니다.

학습 방향

1차 필기에서 리눅스 일반은 깊은 명령 실습보다 용어 구분이 중요합니다. 커널, GNU, 배포판 계열, 라이선스, 디렉터리 역할처럼 이름이 비슷한 개념을 빠르게 구분할 수 있게 정리하세요.

A. 리눅스의 기본 개념

  • 리눅스는 유닉스 계열 운영체제의 특징을 가진 오픈소스 운영체제 계열입니다.
  • 엄밀히 말하면 Linux는 운영체제 전체가 아니라 커널을 의미합니다.
  • 실제 사용 환경은 GNU 도구와 리눅스 커널이 결합된 GNU/Linux 시스템 형태가 많습니다.
  • 대표 특징은 다중 사용자, 다중 작업, 뛰어난 이식성, 안정성, 강력한 네트워크 기능입니다.
  • 서버, 임베디드, 클라우드, 개발 환경 등 다양한 영역에서 사용됩니다.

B. 리눅스의 역사

  • 유닉스는 리눅스의 설계 철학과 명령어 환경에 큰 영향을 준 운영체제입니다.
  • GNU 프로젝트는 자유롭게 사용할 수 있는 유닉스 호환 운영체제 도구를 만들기 위해 시작되었습니다.
  • 리처드 스톨만은 GNU 프로젝트와 자유 소프트웨어 운동의 핵심 인물입니다.
  • 리누스 토르발스는 1991년 리눅스 커널을 공개했습니다.
  • 리눅스 커널은 GPL 라이선스로 공개되어 소스 코드 사용, 수정, 배포가 가능합니다.

C. 오픈소스와 라이선스

오픈소스는 소스 코드를 공개하고 수정·배포를 허용하는 개발 방식입니다. 자유 소프트웨어는 사용, 연구, 수정, 배포의 자유를 강조합니다.

구분핵심 특징시험 포인트
GPL수정·배포 시 파생 저작물도 같은 GPL 조건을 따르도록 요구한다.소스 공개와 copyleft 성격을 기억한다.
BSD저작권 표시 등 조건을 지키면 비교적 자유롭게 재배포할 수 있다.GPL보다 허용 범위가 넓고 상용 제품 포함이 쉽다.

시험에서는 GPL은 공유와 동일 라이선스 유지, BSD는 비교적 느슨한 재배포 허용으로 구분하면 됩니다.

D. 리눅스 배포판

배포판은 커널, GNU 도구, 패키지 관리자, 설치 도구, 기본 프로그램을 묶어 사용하기 쉽게 제공하는 형태입니다.

계열대표 배포판패키지 도구암기 포인트
Debian 계열Debian, Ubuntu, Linux Mintdpkg, apt, apt-getUbuntu는 Debian 계열이다.
Red Hat 계열RHEL, CentOS, Rocky Linux, Fedorarpm, yum, dnfRocky Linux와 Fedora는 Red Hat 계열이다.
Slackware 계열Slackwarepkgtool 등오래된 전통 배포판 계열로 구분한다.
SUSE/openSUSESUSE Linux, openSUSEzypper, YaSTRPM 기반이지만 Red Hat과는 별도 계열로 자주 구분한다.

E. 리눅스 설치와 기본 설정

  • 파티션은 디스크를 논리적으로 나눈 영역입니다.
  • 루트 파티션 `/`은 전체 디렉터리 트리의 시작점입니다.
  • `/home`은 일반 사용자 홈 디렉터리를 분리할 때 자주 사용합니다.
  • swap은 메모리가 부족할 때 보조적으로 사용하는 디스크 영역입니다.
  • 부트로더는 운영체제를 메모리에 적재하는 프로그램이며, 리눅스에서는 GRUB이 대표적입니다.
  • 파일시스템은 파일을 디스크에 저장하고 관리하는 구조이며 ext2, ext3, ext4가 ext 계열입니다.
  • 마운트는 파일시스템을 특정 디렉터리에 연결해 사용할 수 있게 하는 작업입니다.

G. 시험 함정 포인트

  • Linux는 엄밀히 커널을 의미한다.
  • GNU 프로젝트와 리눅스 커널은 구분된다.
  • Ubuntu는 Debian 계열이다.
  • Rocky Linux는 Red Hat 계열이다.
  • /etc는 설정 파일이 위치한다.
  • /home은 일반 사용자 홈 디렉터리이다.
  • /root는 root 사용자의 홈 디렉터리이다.
  • /var는 로그, 메일, 스풀 등 변동 데이터가 저장된다.
  • swap은 메모리 부족 시 보조적으로 사용되는 영역이다.
  • GRUB은 부트로더이다.

H. 예상문제

1. Linux를 엄밀히 설명한 것으로 가장 알맞은 것은?

  1. 셸
  2. 커널
  3. 패키지 관리자
  4. 데스크톱 환경
정답: 2. Linux는 엄밀히 커널을 의미한다.

2. GNU 프로젝트와 가장 관련 깊은 인물은?

  1. 리누스 토르발스
  2. 리처드 스톨만
  3. 빌 조이
  4. 데니스 리치
정답: 2. 리처드 스톨만은 GNU 프로젝트와 자유 소프트웨어 운동의 핵심 인물이다.

3. 리눅스 커널을 공개한 인물은?

  1. 리처드 스톨만
  2. 리누스 토르발스
  3. 브람 몰레나르
  4. 켄 톰프슨
정답: 2. 리누스 토르발스가 1991년 리눅스 커널을 공개했다.

4. Ubuntu가 속한 배포판 계열은?

  1. Debian 계열
  2. Red Hat 계열
  3. Slackware 계열
  4. BSD 계열
정답: 1. Ubuntu는 Debian 계열 배포판이다.

5. Red Hat 계열 패키지 도구로 알맞은 것은?

  1. dpkg
  2. apt-get
  3. rpm
  4. pacman
정답: 3. rpm, yum, dnf는 Red Hat 계열에서 중요하게 다룬다.

6. 시스템 설정 파일이 주로 위치하는 디렉터리는?

  1. /home
  2. /etc
  3. /boot
  4. /dev
정답: 2. /etc는 시스템과 서비스 설정 파일이 위치한다.

7. root 사용자의 홈 디렉터리는?

  1. /home/root
  2. /usr/root
  3. /root
  4. /etc/root
정답: 3. /root는 관리자 계정 root의 홈 디렉터리이다.

8. 부트로더에 해당하는 것은?

  1. GRUB
  2. GNOME
  3. Bash
  4. Apache
정답: 1. GRUB은 리눅스에서 대표적으로 사용되는 부트로더이다.
