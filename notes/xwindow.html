X 윈도우 및 활용

X Window System, X server/client, X.Org, DISPLAY, 윈도 매니저와 데스크톱 환경을 정리합니다.

시험에 나오는 핵심

  • X Window System은 유닉스/리눅스 계열의 그래픽 표시 시스템입니다.
  • X server와 X client의 위치를 반대로 고르는 문제가 자주 나옵니다.
  • GNOME, KDE, Xfce는 데스크톱 환경으로 묶어 기억합니다.

강의식 개념노트

X Window는 GUI를 표시하는 체계입니다

리눅스 자체는 명령줄만으로도 사용할 수 있습니다. 그래픽 화면, 창, 마우스 입력 같은 GUI를 제공하는 대표 체계가 X Window System입니다. X.Org는 그 구현체로 시험에 자주 등장합니다.

server와 client를 반대로 생각해야 합니다

일반적인 웹에서는 서버가 멀리 있고 클라이언트가 내 컴퓨터입니다. X Window에서는 화면과 입력 장치를 가진 쪽이 X server입니다. 그래픽 프로그램은 X client입니다. 이 이름 뒤집기가 이 단원의 핵심 함정입니다.

윈도 매니저와 데스크톱 환경은 범위가 다릅니다

윈도 매니저는 창의 위치, 크기, 테두리, 포커스를 관리합니다. 데스크톱 환경은 윈도 매니저뿐 아니라 패널, 파일 관리자, 설정 프로그램까지 포함합니다. GNOME, KDE, Xfce는 데스크톱 환경입니다.

DISPLAY는 그래픽 출력 주소입니다

DISPLAY 환경변수는 GUI 프로그램이 어느 X server로 화면을 보낼지 지정합니다. 원격 접속 상태에서 GUI 프로그램을 실행할 때 DISPLAY 설정이 맞지 않으면 창이 뜨지 않는 식으로 이해하면 됩니다.

키워드 해설 노트

X server와 X client

X에서 server는 화면·키보드·마우스를 가진 사용자 쪽입니다. client는 화면에 창을 띄우는 응용 프로그램입니다. 이름 때문에 원격 서버와 반대로 착각하는 문제가 자주 나옵니다.

DISPLAY

DISPLAY는 X client가 어느 X server에 화면을 보낼지 지정하는 환경변수입니다. 원격에서 GUI 프로그램을 실행해도 DISPLAY가 내 화면을 가리키면 내 쪽에 창이 뜰 수 있습니다.

startx와 X.Org

startx는 콘솔 환경에서 X 세션을 시작할 때 연결되는 명령입니다. X.Org는 리눅스에서 널리 쓰이는 X Window System 구현체입니다.

윈도 매니저와 데스크톱 환경

윈도 매니저는 창의 이동, 크기 조절, 테두리, 포커스를 담당합니다. 데스크톱 환경은 패널, 파일 관리자, 설정 도구까지 포함한 통합 환경입니다.

원격 X프로그램은 원격에서 실행되어도 화면 출력은 사용자의 X server로 보낼 수 있습니다.

시험 함정 포인트

  • X server는 서버 컴퓨터가 아니라 사용자의 화면·키보드·마우스가 있는 쪽입니다.
  • X client는 그래픽 응용 프로그램입니다.
  • GNOME/KDE는 윈도 매니저 하나가 아니라 데스크톱 환경으로 보는 것이 안전합니다.
  • DISPLAY는 X 출력 대상 지정 변수입니다.

1분 암기 요약

  • X server=내 화면·입력 장치 쪽, X client=실행되는 응용 프로그램.
  • X.Org=대표 X 구현, DISPLAY=출력 대상.
  • GNOME/KDE/Xfce=데스크톱 환경.

처음 공부하는 사람용 흐름

1. X 구조 뒤집기 주의

X server는 프로그램이 돌아가는 서버가 아니라 사용자 화면·키보드·마우스가 있는 쪽입니다. 이 함정이 가장 중요합니다.

2. X client 이해

X client는 화면에 창을 띄우는 응용 프로그램입니다. 원격에서 실행되어도 내 X server로 화면을 보낼 수 있습니다.

3. 환경변수

DISPLAY는 그래픽 출력 대상을 지정합니다. 원격 X나 GUI 실행 문제에서 자주 등장합니다.

4. 데스크톱 환경

GNOME, KDE, Xfce는 데스크톱 환경입니다. 윈도 매니저보다 더 넓은 통합 환경이라는 점을 기억하세요.

기출형 문제 풀이

문제. X Window System에 대한 설명으로 옳은 것은?

  1. X server는 항상 원격 서버 컴퓨터를 의미한다.
  2. X client는 그래픽 응용 프로그램을 의미한다.
  3. DISPLAY는 명령어 검색 경로를 저장한다.
  4. GNOME과 KDE는 파일 압축 도구이다.
정답: 2번

X client는 응용 프로그램입니다. X server는 사용자의 입출력 장치가 있는 쪽이며, DISPLAY는 X 출력 대상을 지정합니다.
