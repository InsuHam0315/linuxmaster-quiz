패키지·압축·백업

Red Hat/Debian 패키지 도구, tar와 압축 도구, 백업 기본 개념을 정리합니다.

시험에 나오는 핵심

  • Red Hat 계열은 rpm, yum, dnf, Debian 계열은 dpkg, apt와 연결합니다.
  • tar는 파일을 묶는 도구이고 압축은 gzip/bzip2/xz 옵션과 함께 처리합니다.
  • 백업은 장애·삭제·변경에 대비해 복구 가능한 사본을 만드는 개념입니다.

강의식 개념노트

패키지 관리는 설치 도구 체계입니다

리눅스에서 프로그램은 보통 패키지 단위로 설치됩니다. 저수준 도구인 rpm, dpkg는 패키지 파일을 직접 다루고, 고수준 도구인 dnf, yum, apt는 저장소와 의존성까지 처리합니다.

계열과 도구를 묶어서 외우세요

Red Hat 계열은 rpm, yum, dnf입니다. Debian 계열은 dpkg, apt입니다. 시험에서 Ubuntu와 rpm을 연결하거나 RHEL과 apt를 연결하면 틀린 선택지일 가능성이 높습니다.

tar는 압축이 아니라 묶기입니다

tar의 기본 역할은 여러 파일을 하나의 archive로 묶는 것입니다. 압축은 gzip, bzip2, xz 같은 방식이 담당합니다. 그래서 tar -czf는 tar로 묶으면서 gzip 압축까지 함께 처리하는 명령입니다.

백업은 복구 관점으로 봅니다

압축 파일을 만들었다고 항상 좋은 백업은 아닙니다. 백업은 장애가 났을 때 원래 상태로 되돌릴 수 있어야 합니다. 전체 백업, 증분 백업, 차등 백업 같은 용어는 “얼마나 자주, 얼마나 많이 복사하느냐”의 차이로 이해하면 됩니다.

명령어 해설 노트

tar 기본

tar -cf backup.tar dir는 디렉터리를 하나의 tar 파일로 묶습니다. c는 생성, x는 해제, t는 목록 확인, f는 파일명 지정입니다.

압축 옵션

z는 gzip, j는 bzip2, J는 xz입니다. tar -czf file.tar.gz dir는 gzip 압축 tar 생성, tar -xzf file.tar.gz는 해제입니다.

패키지 도구

rpm -qa는 Red Hat 계열에서 설치된 패키지 전체 조회입니다. dpkg -i package.deb는 Debian 계열 패키지 파일 설치입니다. 의존성 처리는 보통 dnf, apt가 담당합니다.

백업과 압축 차이

압축은 용량을 줄이는 기술이고, 백업은 장애나 실수에 대비해 복구 가능한 사본을 만드는 운영 절차입니다.

헷갈리는 비교표

항목의미
tar c/x/v/f생성, 해제, 과정 표시, 파일명 지정입니다. f는 보통 archive 파일명과 같이 씁니다.
tar z/j/Jgzip, bzip2, xz 압축 방식과 연결됩니다.
rpm -qa/-qi/-ql/-e전체 조회, 정보 조회, 파일 목록, 삭제입니다.
dpkg -i/-r설치, 제거입니다.
apt update/upgrade패키지 목록 갱신, 설치된 패키지 업그레이드입니다.

시험 함정 포인트

  • tar 자체는 묶기입니다. 압축은 z, j, J 같은 옵션과 구분합니다.
  • tar -czf는 gzip으로 압축한 tar 파일 생성입니다.
  • tar -xzf는 gzip으로 압축된 tar 파일 해제입니다.
  • apt update는 설치가 아니라 목록 갱신입니다.

1분 암기 요약

  • Red Hat=rpm/yum/dnf, Debian=dpkg/apt.
  • tar 옵션: c 만들기, x 풀기, v 보기, f 파일, z gzip, j bzip2, J xz.
  • update=목록 갱신, upgrade=패키지 판올림.

처음 공부하는 사람용 흐름

1. 계열별 도구

패키지 문제는 배포판 계열과 함께 풉니다. Red Hat 계열은 rpm/yum/dnf, Debian 계열은 dpkg/apt입니다.

2. 저수준과 고수준

rpm, dpkg는 패키지 파일을 직접 다루고, dnf, apt는 저장소와 의존성 처리를 함께 합니다.

3. tar와 압축 분리

tar는 묶기입니다. z, j, J가 붙을 때 gzip, bzip2, xz 압축이 함께 처리됩니다.

4. 백업 감각

백업은 복구 가능한 사본을 만드는 것입니다. 압축은 용량 절감이고, 백업은 복구 목적이라는 차이를 기억하세요.

기출형 문제 풀이

문제. tar -czf backup.tar.gz /etc의 의미로 옳은 것은?

  1. /etc를 gzip 방식으로 압축한 tar 파일로 생성한다.
  2. backup.tar.gz를 xz 방식으로 해제한다.
  3. RPM 패키지 목록을 조회한다.
  4. apt 저장소 목록을 갱신한다.
정답: 1번

c는 생성, z는 gzip, f는 파일명 지정입니다. 해제는 x를 사용합니다.
