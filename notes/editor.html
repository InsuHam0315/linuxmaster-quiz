vi 편집기

vi/Vim의 모드 전환, 저장·종료, 삭제·복사·붙여넣기, 검색과 치환을 정리합니다.

시험에 나오는 핵심

  • vi는 모드 기반 편집기입니다. 현재 모드가 명령 의미를 결정합니다.
  • 일반 명령과 콜론으로 시작하는 ex 명령을 구분해야 합니다.
  • 저장하지 않고 종료하는 명령은 자주 출제됩니다.

강의식 개념노트

vi는 모드만 잡으면 절반은 끝납니다

vi가 어려운 이유는 같은 키라도 현재 모드에 따라 의미가 달라지기 때문입니다. 명령 모드에서는 dd가 줄 삭제이고, 입력 모드에서는 글자가 입력됩니다. 시험에서는 “입력 모드로 전환”, “명령 모드에서 삭제”처럼 모드를 섞어 묻습니다.

입력 시작 키는 위치 차이입니다

i는 커서 앞, a는 커서 뒤, o는 아래 줄을 새로 만들고 입력합니다. 단순 암기보다 커서 기준으로 위치를 상상하면 헷갈리지 않습니다. 입력을 끝내는 키는 항상 ESC입니다.

저장과 종료는 콜론 명령입니다

:w, :q, :wq, :q!는 모두 콜론으로 시작하는 ex 명령입니다. 시험에서는 특히 저장하지 않고 나가는 :q!, 저장 후 종료하는 :wq를 자주 묻습니다.

검색·치환은 형태를 통째로 보세요

/문자열은 검색이고, n과 N으로 결과를 이동합니다. :%s/old/new/g는 전체 문서에서 old를 new로 모두 바꾸는 치환 명령입니다. %는 전체 범위, g는 한 줄 안의 모든 매칭을 뜻한다고 이해하면 됩니다.

핵심 개념 정리

모드/명령정리
명령 모드이동, 삭제, 복사, 붙여넣기 같은 일반 조작을 수행합니다.
입력 모드문자 입력 모드입니다. i, a, o로 들어갑니다.
ex 모드:로 시작하는 저장, 종료, 설정, 치환 명령을 입력합니다.
ESC입력 모드에서 명령 모드로 돌아갑니다.
:w, :q, :wq, :q!저장, 종료, 저장 후 종료, 저장하지 않고 강제 종료입니다.

명령어 해설 노트

입력 모드 진입

i는 커서 앞, a는 커서 뒤, o는 아래에 새 줄을 만들고 입력합니다. 입력을 끝내려면 ESC로 명령 모드에 돌아옵니다.

삭제·복사·붙여넣기

dd는 현재 줄 삭제, yy는 현재 줄 복사, p는 붙여넣기입니다. 앞에 숫자를 붙이면 반복되어 3dd는 세 줄 삭제입니다.

검색과 이동

/word는 아래 방향 검색, ?word는 위 방향 검색입니다. n은 같은 방향 다음 결과, N은 반대 방향 결과로 이동합니다.

저장·종료·치환

:w는 저장, :q는 종료, :wq는 저장 후 종료, :q!는 저장하지 않고 강제 종료입니다. :%s/old/new/g는 전체 치환입니다.

헷갈리는 비교표

명령의미
i / a / o커서 앞 입력, 커서 뒤 입력, 아래 줄 새 줄 입력입니다.
dd / yy / p한 줄 삭제, 한 줄 복사, 붙여넣기입니다.
x / u문자 삭제, 되돌리기입니다.
n / N검색 결과 다음 방향, 반대 방향 이동입니다.
:set number / :set nonumber줄 번호 표시, 줄 번호 숨김입니다.

시험 함정 포인트

  • :q!는 저장하지 않고 종료합니다.
  • dd는 한 줄 삭제, yy는 한 줄 복사입니다.
  • :%s/old/new/g는 전체 범위에서 문자열을 치환합니다.
  • 콜론 명령은 ex 모드, dd 같은 명령은 명령 모드에서 사용합니다.

1분 암기 요약

  • 입력: i 앞, a 뒤, o 아래 새 줄.
  • 저장·종료: :w, :q, :wq, :q!.
  • 편집: dd 삭제, yy 복사, p 붙이기, u 취소.

처음 공부하는 사람용 흐름

1. 모드가 전부

vi는 모드를 모르면 명령이 이해되지 않습니다. 글자를 입력하는 입력 모드와 명령을 내리는 명령 모드를 반드시 구분하세요.

2. 입력 시작 키

i는 커서 앞, a는 커서 뒤, o는 아래 새 줄입니다. 입력을 끝내려면 ESC입니다.

3. 저장과 종료

콜론으로 시작하는 명령은 ex 명령입니다. :wq와 :q!는 거의 매번 나오는 암기 포인트입니다.

4. 검색과 치환

/문자열로 찾고 n, N으로 이동합니다. 전체 치환은 :%s/old/new/g 형태입니다.

기출형 문제 풀이

문제. vi 명령에 대한 설명으로 옳지 않은 것은?

  1. dd는 현재 줄을 삭제한다.
  2. yy는 현재 줄을 복사한다.
  3. :q!는 저장하지 않고 강제 종료한다.
  4. u는 아래 줄에 새 줄을 만들고 입력 모드로 전환한다.
정답: 4번

u는 되돌리기입니다. 아래 줄을 만들고 입력 모드로 들어가는 키는 o입니다.
