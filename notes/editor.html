vi 편집기

vi/Vim의 모드 전환, 저장·종료, 삭제·복사·붙여넣기, 검색과 치환을 정리합니다.

시험에 나오는 핵심

  • vi는 모드 기반 편집기입니다. 현재 모드가 명령 의미를 결정합니다.
  • 일반 명령과 콜론으로 시작하는 ex 명령을 구분해야 합니다.
  • 저장하지 않고 종료하는 명령은 자주 출제됩니다.

핵심 개념 정리

모드/명령정리
명령 모드이동, 삭제, 복사, 붙여넣기 같은 일반 조작을 수행합니다.
입력 모드문자 입력 모드입니다. i, a, o로 들어갑니다.
ex 모드:로 시작하는 저장, 종료, 설정, 치환 명령을 입력합니다.
ESC입력 모드에서 명령 모드로 돌아갑니다.
:w, :q, :wq, :q!저장, 종료, 저장 후 종료, 저장하지 않고 강제 종료입니다.

자주 나오는 명령어/키워드

  • i
  • a
  • o
  • dd
  • yy
  • p
  • x
  • u
  • /검색어
  • n
  • N
  • :set number
  • :%s/old/new/g

헷갈리는 비교표

명령의미
i / a / o커서 앞 입력, 커서 뒤 입력, 아래 줄 새 줄 입력입니다.
dd / yy / p한 줄 삭제, 한 줄 복사, 붙여넣기입니다.
x / u문자 삭제, 되돌리기입니다.
n / N검색 결과 다음 방향, 반대 방향 이동입니다.
:set number / :set nonumber줄 번호 표시, 줄 번호 숨김입니다.

시험 함정 포인트

  • :q!는 저장하지 않고 종료합니다.
  • dd는 한 줄 삭제, yy는 한 줄 복사입니다.
  • :%s/old/new/g는 전체 범위에서 문자열을 치환합니다.
  • 콜론 명령은 ex 모드, dd 같은 명령은 명령 모드에서 사용합니다.

1분 암기 요약

  • 입력: i 앞, a 뒤, o 아래 새 줄.
  • 저장·종료: :w, :q, :wq, :q!.
  • 편집: dd 삭제, yy 복사, p 붙이기, u 취소.
