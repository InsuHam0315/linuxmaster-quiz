리눅스마스터 2급 2차 연계 포인트

기말고사 범위와 겹치는 리눅스마스터 2급 2차 기출·복원 유형의 출제 포인트만 선별합니다.

연계

1. 단원 개요

이 페이지는 자격증 전체 대비가 아니라 운영체제 기말 범위와 겹치는 리눅스 실습형 문제만 모아 보는 용도입니다.

관련 CBT 풀기 리눅스마스터 연계 보기

2. 기말고사에서 중요한 이유

리눅스마스터 2급 2차는 명령어 옵션, 권한 숫자, 프로세스, 패키지, 네트워크, systemd 같은 객관식 함정이 많아 기말 CBT 유형 설계에 유용합니다.

3. 리눅스마스터 2급 2차 기출 연계 포인트

  • 파일/디렉터리 명령 옵션
  • chmod/chown/umask와 특수권한
  • ps/top/kill/jobs
  • bash export/PATH/리다이렉션
  • vi 저장·종료·검색·치환
  • apt/dpkg/rpm/yum/dnf
  • tar/gzip/bzip2/xz
  • mount/LVM/fstab
  • 네트워크 포트와 진단 명령
  • GRUB/systemd target

4. 핵심 개념 설명

  • 원문 문제를 외우기보다 반복되는 개념과 틀리기 쉬운 선지를 정리합니다.
  • 기말 범위 밖인 X 윈도우, 응용 분야, 일부 멀티미디어 영역은 수업자료에 없으면 제외합니다.
  • 명령어는 이름, 옵션, 대상 순서를 함께 외워야 합니다.
  • 보기에서 비슷한 명령을 섞는 방식이 자주 사용됩니다.

7. 시험 함정 포인트

  • 원문 문제 암기가 목적이 아닙니다.
  • 자격증 전체 범위가 아니라 기말 범위와 겹치는 항목만 봅니다.
  • 명령어 옵션 하나 차이로 정답이 갈립니다.
  • start와 enable, update와 upgrade, c와 x를 특히 구분합니다.

9. 예상문제

1. 기말 범위와 가장 직접적으로 겹치는 리눅스마스터 유형은?

  1. ① X 윈도우 역사만 암기
  2. ② chmod와 프로세스 관리 명령
  3. ③ 그래픽 디자인 도구 비교
  4. ④ 프린터 모델명 암기
정답: 2. 권한과 프로세스 명령은 기말 실습 범위와 직접 겹칩니다.

2. systemctl enable의 의미는?

  1. ① 즉시 시작만 수행
  2. ② 부팅 시 자동 시작 등록
  3. ③ 파일 압축
  4. ④ DNS 질의
정답: 2. enable은 자동 시작 등록입니다.
