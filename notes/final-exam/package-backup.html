패키지·압축·백업

apt/dpkg, rpm/yum/dnf, tar, gzip, bzip2, xz, zip, 백업 명령을 정리합니다.

패키지

1. 단원 개요

패키지 관리는 소프트웨어 설치와 업데이트를 담당하고, 압축·아카이브는 여러 파일을 묶거나 보관하는 데 사용됩니다.

관련 CBT 풀기 리눅스마스터 연계 보기

2. 기말고사에서 중요한 이유

기말 범위의 apt update/upgrade/install, systemctl status, tar 옵션 c/x/t/z/j/J가 자주 출제됩니다.

3. 리눅스마스터 2급 2차 기출 연계 포인트

  • apt update와 upgrade 차이
  • dpkg와 apt 차이
  • rpm -qa/-e/-Uvh
  • tar c/x/t 옵션
  • gzip/bzip2/xz 옵션 구분

4. 핵심 개념 설명

  • apt는 저장소 기반으로 의존성을 처리합니다.
  • dpkg와 rpm은 개별 패키지 파일 관리에 가깝습니다.
  • tar는 압축 자체보다 여러 파일을 하나의 아카이브로 묶는 역할이 핵심입니다.
  • gzip은 z, bzip2는 j, xz는 J 옵션으로 연결해 외우면 좋습니다.

5. 주요 명령어

명령어핵심 의미

6. 헷갈리는 비교표

구분핵심암기 포인트

dpkg/rpm개별 패키지 기반의존성 수동 처리 가능
tar -c생성create
tar -x해제extract
tar -t목록table/list

7. 시험 함정 포인트

  • apt update는 설치가 아니라 목록 갱신입니다.
  • rpm -e는 제거, -qa는 전체 조회입니다.
  • tar -czf는 생성, -xzf는 해제입니다.
  • bzip2는 -j, xz는 -J입니다.

8. 자주 나오는 선지 표현

  • 다운로드한 .deb 파일을 설치하는 명령은?
  • tar.gz를 해제하는 옵션은?
  • RPM 전체 패키지 조회 명령은?

9. 예상문제

1. gzip tar 파일을 해제하는 명령은?

  1. ① tar -czf file.tar.gz
  2. ② tar -xzf file.tar.gz
  3. ③ tar -jtvf file.tar.gz
  4. ④ gzip -c file.tar.gz
정답: 2. -x는 해제, -z는 gzip입니다.

2. 설치된 RPM 전체 목록 조회는?

  1. ① rpm -qa
  2. ② rpm -e
  3. ③ rpm -Uvh
  4. ④ rpm -i
정답: 1. q=query, a=all입니다.

10. 최종 암기 요약

apt update = 목록 갱신
apt install = 설치
dpkg -i = deb 설치
rpm -qa = 전체 조회
tar c/x/t, gzip z, bzip2 j, xz J
