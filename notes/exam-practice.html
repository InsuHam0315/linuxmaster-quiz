기출형 실전 문제 풀이

실제 원문을 복제하지 않고, 리눅스마스터 2급 필기 출제 방식에 맞춰 직접 만든 실전형 문제입니다.

저작권 안내

  • 공개 기출 원문을 그대로 옮기지 않습니다.
  • 아래 문제는 시험 범위와 자주 나오는 함정 포인트를 바탕으로 새로 작성한 기출형 문제입니다.
  • 실제 원문 문제는 KAIT 또는 공식 제공 경로에서 확인하는 것이 안전합니다.

기출형 실전 문제 1

문제. 다음 중 리눅스 파일 권한과 특수권한에 대한 설명으로 가장 옳은 것은?

  1. chmod 1777 /shared에서 앞자리 1은 setuid를 의미한다.
  2. 디렉터리의 실행 권한 x는 해당 디렉터리 안으로 접근하거나 하위 경로를 탐색할 수 있음을 의미한다.
  3. chmod 2750 project에서 앞자리 2는 sticky bit를 의미한다.
  4. 파일의 읽기 권한 r은 해당 파일을 실행할 수 있음을 의미한다.
정답: 2번

디렉터리에서 x는 파일 실행이 아니라 접근·탐색 권한입니다. 이 권한이 없으면 디렉터리 안 파일명을 알고 있어도 접근이 막힐 수 있습니다.

선지별 해설

  • 1번: 앞자리 1은 sticky bit입니다. setuid는 4입니다.
  • 2번: 디렉터리 x의 핵심 의미를 정확히 설명합니다.
  • 3번: 앞자리 2는 setgid입니다. sticky bit는 1입니다.
  • 4번: 파일 r은 읽기 권한이고 실행은 x입니다.

시험장에서 이렇게 풀기

  • 특수권한 숫자는 4=setuid, 2=setgid, 1=sticky 순서로 먼저 적습니다.
  • 파일과 디렉터리의 x 의미를 반드시 분리합니다.
  • /tmp 같은 공유 디렉터리는 sticky bit 예시로 자주 연결됩니다.

추가 연습 문제

문제. 다음 명령의 의미로 가장 알맞은 것은?

find /var/log -type f -name "*.log" -exec grep -i "error" {} \;

  1. /var/log 아래의 모든 디렉터리 이름을 error로 변경한다.
  2. /var/log 아래의 일반 파일 중 확장자가 .log인 파일에서 대소문자 구분 없이 error를 검색한다.
  3. /var/log 아래의 .log 파일을 모두 삭제한다.
  4. 표준에러를 표준출력으로 합쳐 파일에 저장한다.
정답: 2번

-type f는 일반 파일, -name "*.log"는 이름 조건, -exec ... {} \;는 찾은 파일마다 명령을 실행한다는 뜻입니다. grep -i는 대소문자를 무시합니다.
