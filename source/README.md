# source 자료 및 웹 참고 출처

## 로컬 자료 반영 방식

- `source/notion/`: 사용자가 노션에 정리한 운영체제 기말고사 필기 자료로, 기말 CBT와 강의노트의 1순위 기준으로 사용했다.
- `source/lecture-pdf/`: 교수님 수업 PDF로, 파일시스템, 부팅, 패키지, 사용자/그룹, 네트워크 범위의 개념 정확도 확인 기준으로 사용했다.
- 로컬 원본 PDF 내용은 웹 페이지에 그대로 복사하지 않고, 학습용 요약·재구성 문장으로 반영했다.

## 저작권 및 사용 원칙

- 공개적으로 접근 가능한 자료만 참고했다.
- 로그인, 유료 자료, 캡차 우회, 차단 우회, 대량 크롤링은 하지 않았다.
- 원문 문제를 대량 복제하지 않고, 개념·유형·선지 함정을 바탕으로 새 문장과 해설을 작성했다.
- 운영체제 기말 CBT는 기말고사 범위와 겹치는 리눅스마스터 2급 2차 유형만 우선 반영했다.
- 별도 리눅스마스터 2급 2차 CBT는 공식 시험 구조와 공개 CBT 유형을 기반으로 하되, 실제 시험 원문 대체물이 아니라 학습용 재구성 문제다.
