# source 자료 및 웹 참고 출처

## 로컬 자료 반영 방식

- `source/notion/`: 사용자가 노션에 정리한 운영체제 기말고사 필기 자료로, 기말 CBT와 강의노트의 1순위 기준으로 사용했다.
- `source/lecture-pdf/`: 교수님 수업 PDF로, 파일시스템, 부팅, 패키지, 사용자/그룹, 네트워크 범위의 개념 정확도 확인 기준으로 사용했다.
- 로컬 원본 PDF 내용은 웹 페이지에 그대로 복사하지 않고, 학습용 요약·재구성 문장으로 반영했다.

## 저작권 및 사용 원칙

- 공개적으로 접근 가능한 자료만 참고했다.
- 로그인, 유료 자료, 캡차 우회, 차단 우회, 대량 크롤링은 하지 않았다.
- 운영체제 기말 CBT는 기말고사 범위와 겹치는 리눅스마스터 2급 2차 유형만 우선 반영하고, 문항은 기말 학습용으로 재구성했다.
- 별도 리눅스마스터 2급 2차 CBT는 사용자가 날짜별 공개 기출 풀이를 원한 범위에 한해 ComCBT 공개 페이지를 출처로 명시하고, 2023-09-09 및 2023-12-09 회차를 기말 CBT와 분리된 날짜별 기출 학습 화면으로 제공한다.
- 이미지·표가 필요한 문제는 원본 이미지를 저장하지 않고 공개 URL을 `passage`에 표시해 원본 자료를 확인할 수 있게 했다.
